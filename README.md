**Advanced Insurance Policy Data Analyzer**
This project contains a Python-based policy data analyzer with advanced filtering, statistical analysis, anomaly detection, and automated visualizations.

Features:
- Interactive filtering by insurance type, company, policy age, premium range, and claim status
- Automated data quality checks and anomaly detection
- Dynamic visualizations (charts, graphs, time series analysis)
- Comprehensive metrics reports with AI-generated insights
- Multi-format export (Excel with pivot tables, CSV)
